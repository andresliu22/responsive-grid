CSS Snippet Cheatsheet

Flexbox Row

Use these three properties...

.row {
    display: flex;
    flex-direction: row;
    flex-wrap: wrap;
}
Flexbox Column

Use these three properties...

.column {
    display: flex;
    flex-direction: column;
}
CSS Grid Layout

Use these three properties...

.grid {
    display: grid;
    width: 100%;
    grid-template-columns:
repeat(12,1fr);
}
Linear Gradients

Use these three properties...

.linear-gradient-background {
    background-image: linear-
gradient(
        rgba(232, 102, 236, 0.3) 0%,
        rgba(232, 102, 236, 0.9) 100%
    );
}
Box Transition Glow

Use these three properties...

.code-card .card-header {
    border-radius: 8px;
    transition: all 0.5s ease-in-out;
}

.code-card:hover,
.code-card:hover .card-header {
    box-shadow: inset 0px 0px 8px
    rgba(232, 201, 236, 1), 0 0 15px
    rgba(232, 201, 236, 1);
}
Overlay Card with Title

Use these three properties...

.card-header {
    position: relative;
    margin-top: -20px;
}
